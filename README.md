# مجموعه برنامه‌های وب ورزشی — سیستان و بلوچستان

> **سازنده:** امین عزیزی مقدم  
> **سازمان:** اداره کل ورزش و جوانان استان سیستان و بلوچستان

مجموعه‌ای از ۱۲ برنامه وب تک‌فایلی (Single-Page HTML) برای مدیریت و مستندسازی فعالیت‌های ورزشی استان. هر فایل به‌صورت مستقل در مرورگر اجرا می‌شود — بدون نیاز به server یا نصب.

## نحوه استفاده

فایل HTML موردنظر را در مرورگر باز کنید. داده‌ها به‌صورت خودکار در `localStorage` مرورگر ذخیره می‌شوند.

**قابلیت‌های مشترک:**
- ورود/خروج داده از فرمت Excel
- خروجی PDF، Word، Excel
- تقویم جلالی (شمسی)
- پشتیبانی از چاپ
- تم روشن/تیره
- پشتیبان‌گیری و بازگردانی داده
