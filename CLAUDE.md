## ساختار مخزن

هر پروژه یک فایل `‎.html` مستقل است که کد HTML, CSS و JavaScript آن درون یک فایل قرار دارد. هیچ build step، bundler یا dependency خارجی‌ای وجود ندارد — فایل را در مرورگر باز کنید و کار می‌کند.

```
/
├── CLAUDE.md
├── README.md
├── سامانه_آمار_ورزشی_v8_نهایی.html   ← پروژه ۱ (فعلی)
├── [پروژه ۲].html
├── [پروژه ۳].html
└── ...                                 ← تا ۱۱ پروژه
```

## الگوهای مشترک فنی

### ذخیره‌سازی داده
- تمام داده‌ها در **`localStorage`** مرورگر ذخیره می‌شوند (بدون backend).
- کلیدها معمولاً به صورت `sports_data_v*` یا مشابه نام‌گذاری شده‌اند.
- برای undo/redo از **snapshot** درون localStorage استفاده می‌شود.

### تقویم
- تقویم **جلالی (شمسی)** با picker سفارشی درون‌فایلی.
- تبدیل عدد با `toFarsiDigits()` / `toArabicDigits()`.

### ورودی/خروجی اکسل
- کتابخانه **XLSX-Lite** سفارشی (بدون وابستگی خارجی) درون هر فایل HTML.
- خروجی Excel/Word/PDF با `exportPageExcel()` / `exportPageWord()` / `exportPagePDF()`.
- ورودی اکسل با پیش‌نمایش و انتخاب شیت: `importExcelForPage()`.

### رابط کاربری
- فونت: **Vazirmatn** از Google Fonts.
- CSS Variables برای تم روشن/تیره.
- پشتیبانی کامل از **چاپ** با `@media print`.
- نمودارها: canvas-based (bar chart, pie chart) — بدون Chart.js.

### امنیت
- تغییر رمز عبور ساده با `changePassword()` — داده‌ها **رمزنگاری نمی‌شوند**.
- هیچ ارتباط شبکه‌ای وجود ندارد؛ همه چیز سمت کلاینت است.

## دستورالعمل برای Claude

### هنگام ویرایش فایل‌های HTML
- فایل‌ها بزرگ هستند (۳۰۰۰+ خط). فقط بخش موردنظر را ویرایش کن، کل فایل را بازنویسی نکن.
- CSS در `<style>` تگ، JS در `<script>` تگ — هر دو درون همان فایل.
- اعداد فارسی را با `toFarsiDigits()` نمایش بده.
- اسامی توابع/متغیرها به انگلیسی، متون UI به فارسی.

### افزودن پروژه جدید
1. یک فایل `.html` جدید با نام فارسی در ریشه بساز.
2. جدول «پروژه‌های موجود» در همین فایل CLAUDE.md را به‌روز کن.
3. README.md را نیز به‌روز کن.

### چیزهایی که نباید تغییر کنی
- ساختار XLSX-Lite سفارشی — کتابخانه اختصاصی است.
- منطق snapshot/history — کاربر به بازگردانی وابسته است.
- نام توابع `export*` و `import*` — ممکن است در جاهای دیگر ارجاع داشته باشند.
